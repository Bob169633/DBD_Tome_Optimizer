# DBD Tome Optimizer
The goal of the program was to graph the challenges shown on each page of the DBD tome and display which challenges must be claimed in order to maximize the amount of blood points unclaimed to bank them when chapters and characters are released.

## Installing
Review the code, then navigate to releases and download the .exe listed there
All data files are attached in the repository for review if you're paranoid about just downloading an executable
